Simulations of the effect of vaccinating children of age 12-15 in Sweden.

R-code for the technical report found here:
https://www.folkhalsomyndigheten.se/publicerat-material/publikationsarkiv/c/covid-19-vaccination-av-barn-fran-12-ar-i-sverige--beslutsunderlag-september-2021/

Code can be found in folder Script, data used in simulation can be found in folder Data, and the results are saved in folder Output.
